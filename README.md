# Feature Engineering Capstone Assignment

## Objective
This project focuses on feature engineering, preprocessing, scaling, feature selection, and model evaluation using machine learning techniques.

## Files Included
- FeatureEngineering_Capstone.ipynb
- requirements.txt
- report/Report.pdf

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Tasks Completed
- Data loading and exploration
- Missing value handling
- Encoding categorical features
- Numeric preprocessing
- Feature scaling
- Feature selection
- KNN model training
- Random Forest model training
- Accuracy comparison
- ROC-AUC evaluation

## How to Run
1. Install libraries from requirements.txt
2. Open the notebook
3. Run all cells from top to bottom
